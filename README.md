# Customer Segmentation using K-Means Clustering
This project focuses on customer segmentation for a retail store using the K-Means Clustering algorithm, an unsupervised machine learning technique. The main objective of this project is to analyze customer purchase behavior and divide customers into different groups based on their Annual Income and Spending Score.
Customer segmentation helps businesses better understand their customers and create effective marketing strategies. By grouping customers with similar spending patterns, companies can improve customer targeting, personalized marketing, product recommendations, and overall business decision-making.
In this project, the Mall Customer Dataset is used to perform clustering analysis. The Elbow Method is applied to determine the optimal number of clusters for the dataset. After identifying the appropriate number of clusters, the K-Means algorithm is implemented to classify customers into different segments.
The project also includes data visualization using Matplotlib to clearly represent customer clusters and cluster centroids. Different colors are used to distinguish each customer group, making the analysis easy to understand and interpret.

## Features
- Customer segmentation using K-Means Clustering
- Data preprocessing and feature selection
- Elbow Method for optimal cluster selection
- Data visualization of customer groups
- Cluster centroid identification

## Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Outcome
The final output of the project is a visual representation of customer segments that helps identify different categories of customers based on their purchasing behavior. This analysis can support businesses in improving marketing strategies and customer engagement.
